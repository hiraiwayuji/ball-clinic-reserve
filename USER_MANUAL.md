# 接骨院 DX 経営管理ツール 導入・運用マニュアル

## 桃 コンセプト
### 「AI 軍師」と歩む、次世代の接骨院経営
このシステムは、単なる予約管理ツールではありません。
先生を日々の単純作業（レシート整理やSNS投稿のネタ探し）から解放し、**「経営そのものを楽しむ」ための強力なパートナー**となることを目指しています。

---

## 1. システムの起動方法
毎朝、PCを開いたらデスクトップにあるアイコンをクリックするだけで準備完了です。

1.  **起動**: デスクトップの「**接骨院ダッシュボード起動.bat**」をダブルクリックします。
2.  **待機**: サーバーが自動で立ち上がり、約15秒後にブラウザで管理画面が開きます。
3.  **ログイン**: 登録した管理者メールアドレスとパスワードでログインしてください。

---

## 2. 主要機能ガイド

### 督 ダッシュボード（経営の可視化）
ログイン後、最初に表示される画面です。院の「今」がひと目で分かります。
- **売上目標進捗**: 今月の目標に対して現在いくら売上が上がっているかをリアルタイムで表示。
- **予約タイムライン**: 今日の予約状況を時系列で確認。
- **AIタスク**: AIが今日の状況に基づき、「今日やるべき優先事項」を提案します。

### 珸・ 顧客管理 ＆ LINE連携
患者さん一人ひとりの詳細な情報を管理し、LINEを通じてファン化を促進します。
- **LINE自動連携**: 患者さんがLINEでメッセージを送るだけで、システム内のカルテと自動紐づけ。
- **属性分析**: 年齢層、地域、来院経路（紹介・Web等）をチャートで分析可能。
- **誕生日クーポン**: 誕生月の患者さんへLINEでメッセージやクーポンを自動案内。

### 債 経費入力フロー（レシート管理）
「忙しくてレシートが溜まってしまう」という問題を解決するための特化機能です。
1.  **とりあえず保存**: スマホ等でレシートの写真を撮り、「とりあえず保存」ボタンで一時保存。
2.  **後で仕分け（トリアージ）**: 手が空いた時に、写真を見ながらカテゴリ（消耗品、通信費など）を仕分けし、正式登録。
3.  **不明・保留**: 判断に迷うものは「保留」リストに残し、月次処理などでまとめて確認できます。

### 錐 AI軍師（経営アドバイス ＆ ブログ提案）
最先端の AI（Gemini）が、あなたの院のデータを分析してアドバイスします。
- **軍師メモ**: 経営上の気づきや AI への指示をメモしておくと、AI がそれを踏まえて回答します。
- **週刊ブログ提案**: 院の最近の出来事やトレンドに基づき、**「note」やブログのタイトル・構成案を週に1回自動提案**します。

## 3. 日々の運用スケジュール（例）

| 時間帯 | アクション | ツールの活用 |
| :--- | :--- | :--- |
| **朝（始業前）** | 今日の状況確認 | ダッシュボードで予約数と「AI推奨タスク」をチェック。 |
| **日中（診療中）** | 売上・経費入力 | 窓口売上をその場で入力。レシートは写真に撮って「とりあえず保存」。 |
| **夜（終業後）** | 振り返り ＆ 仕分け | 溜まった保留経費を仕分け。AI軍師のアドバイスを確認し、明日への指針に。 |
| **週1回** | Web集客準備 | AIが提案したブログ案をもとに、SNSやブログを更新。 |

---

## 4. セキュリティと安心設計
- **個人情報の保護**: 患者データは先生ご自身のクラウドアカウント（Supabase）にのみ保存されます。
- **アクセス制限**: 管理者以外は管理画面にアクセスできないよう厳重な認証ガードを設けています。
- **リアルタイム同期**: 複数のPCやスマホで開いても、データは常に最新の状態に同期されます。
